## 2. 官网页面清单

### 2.1 首页 `/`

#### 页面目标

- 体现公司品牌和专业可信度
- 快速说明两条核心业务线
- 引导客户进入报价或联系

#### 页面区块

- 顶部导航
- Hero 主视觉
- 公司简介
- 核心业务总览
- 技术服务分类概览
- 代采业务介绍
- 合作流程
- 常见问题
- 联系方式
- Footer

#### 关键 CTA

- 查看技术服务
- 进入报价中心
- 立即咨询

### 2.2 技术服务 `/services`

#### 页面区块

- 页面头图与概述
- 服务分类导航
- 分类介绍区
- 服务流程
- 交付说明
- 常见问题
- 询价 CTA

#### 与后台对应内容

- 分类名称
- 分类说明
- 展示顺序
- 介绍文案
- 图文素材

### 2.3 试剂耗材代采 `/procurement`

#### 页面区块

- 业务介绍
- 可代采品类
- 合作平台说明
- 服务流程
- 开票与交付说明
- 询价 CTA

#### 页面重点

- 强调不是商城，而是商务代采能力展示
- 强调可配合高校、医院、科研机构采购流程

### 2.4 报价中心 `/quote`

#### 页面区块

- 顶部品牌区
- 报价说明
- 搜索区
- 分类筛选区
- 项目筛选区
- 价格筛选区
- 服务条目列表
- 报价单侧边栏/弹窗
- 客户信息表单

#### 页面能力

- 加入报价
- 修改数量
- 移除项目
- 提交询价
- 复制询价内容

### 2.5 关于我们 `/about`

#### 页面区块

- 公司介绍
- 业务定位
- 服务对象
- 合作方式
- 联系方式

### 2.6 联系我们 `/contact`

#### 页面区块

- 联系方式展示
- 公司地址
- 电话/邮箱
- 留言表单

## 5. 页面与业务实体映射

| 页面 | 后端模块 | 主要实体 |
|---|---|---|
| 首页 | CMS | pages, banners, contact_info |
| 技术服务 | Service Catalog | service_categories, service_projects |
| 代采页面 | CMS + Procurement | pages, supplier_platforms |
| 报价中心 | Quote + Service Catalog | service_items, quote_requests |
| 询价管理 | Quote | quote_requests, quote_request_items |
| 订单管理 | Order | orders, order_items |
| 客户管理 | CRM | customers, contacts, invoice_profiles |
| 合同管理 | Contract | contracts |
| 供应商链接管理 | Procurement | supplier_links, procurement_lists |

## 6. 内容运营建议

- 首页和业务页面要支持后台可配置，避免每次改文案都发版。
- 服务目录数据和展示文案要分离，避免价格调整影响官网介绍内容。
- 公司基本信息单独维护，供官网页脚、联系页、导出模板复用。
